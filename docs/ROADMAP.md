# Roadmap

## Phase 1: Core Foundation
- Typed action schema and wrapper registry.
- Deterministic state machine + trace.
- Invariant/precondition/budget enforcement.

## Phase 2: Formal Depth
- SMT-backed verification engine.
- Temporal runtime monitor.
- Bounded model checking and counterexample reporting (normalized witness payloads for Phase 3 ingestion).

## Phase 3: Integrations & Tooling
- First-party LangGraph/OpenAI adapters.
- Verification report certificate pipeline consuming canonical Phase 2 report contracts.
- Replay and forensic tooling.

## Phase 4: Enterprise Hardening
- Multi-agent policy model.
- Advanced info-flow and sandboxing.
- Compliance mappings and third-party audit support.
